\# Projet AlgoBox – Travail collaboratif



\## Description

Ce dépôt contient un projet AlgoBox réalisé en binôme dans un cadre scolaire.  

Les algorithmes sont développés avec le logiciel AlgoBox et partagés via GitHub afin de permettre un travail collaboratif et un suivi des modifications.

\## Organisation du dépôt

\- `src/` : contient l’ensemble des fichiers AlgoBox du projet (fichiers `.algo`)

\- `README.md` : explique comment ouvrir et exécuter les algorithmes



\## Prérequis

Avant d’ouvrir le projet, il est nécessaire de disposer de :

\- AlgoBox installé sur l’ordinateur

\- Git (optionnel, uniquement si le projet est cloné depuis GitHub)



\## Comment ouvrir les fichiers AlgoBox

1\. Télécharger ou cloner le dépôt GitHub sur votre ordinateur.

2\. Ouvrir le logiciel \*\*AlgoBox\*\*.

3\. Cliquer sur \*\*Fichier → Ouvrir\*\*.

4\. Aller dans le dossier `src/` du projet.

5\. Sélectionner le fichier `.algo` que vous souhaitez ouvrir.



\## Comment exécuter un algorithme

1\. Une fois le fichier `.algo` ouvert dans AlgoBox, vérifier que le programme est complet.

2\. Cliquer sur le bouton \*\*Exécuter\*\* (ou utiliser la touche prévue par AlgoBox).

3\. Suivre les instructions affichées à l’écran (saisies utilisateur si demandées).

4\. Observer le résultat dans la fenêtre d’exécution.



\## Remarques

\- Chaque fichier `.algo` correspond à un algorithme indépendant.

\- Les fichiers peuvent être modifiés puis enregistrés directement depuis AlgoBox.

\- Les versions finales du projet sont disponibles via un tag GitHub.
